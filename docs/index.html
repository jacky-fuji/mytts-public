ローカル日本語TTS Voice Clone 音声サンプル

Zenn記事「日本語ボイスクローン6モデル比較: Irodori-TTS/Qwen3-TTS/Fish Speech等を音響分析」の補助ページです。同一文のモデル比較、公開用に選抜した生成音声、Irodori-TTS 600M VoiceDesignで生成したEV発表スピーチ抜粋を試聴できます。

Zenn記事を読む GitHubリポジトリを見る
この公開版では、プライバシー保護のため本人の元録音は含めていません。音声サンプルは公開用に選抜した生成音声を、元WAVからMP3へエンコードし、試聴しやすい音量へ正規化したものです。
公開用音声は、生成されたWAVファイルを元に、FFmpegの loudnorm=I=-18:TP=-1.5:LRA=11 で試聴音量をそろえ、MP3 160kbpsへエンコードしています。分析本文で扱った音響指標は、原則として元WAVを対象にしています。
